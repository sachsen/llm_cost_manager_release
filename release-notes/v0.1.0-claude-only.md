# Agent Meter for Claude Code v0.1.0

Claude Codeだけを対象にしたWindows x64版です。

## 主な機能

- Claude Codeのモデル、コンテキスト、ローカルトークン統計
- 期間別グラフ
- Claude Codeセッション開始時のタスクトレイ自動起動
- フル版とは異なるポート`127.0.0.1:4319`と専用データ領域

## 専用版で無効な機能

- CodexのAPI通信、ローカル履歴読取、フック設定、表示
- GLMのZ.AI／BigModel通信、設定・認証値読取、履歴集計、表示
- GitHub Copilotの検出と表示
- Claude Code上でモデル名が`GLM`から始まるセッションの集計

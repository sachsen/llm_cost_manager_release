# Agent Meter v0.1.0

最初の公開プレビューです。

## 主な機能

- Codexの契約枠とローカルトークン統計
- Claude Codeのモデル、コンテキスト、ローカルトークン統計
- GLM Coding Planの5時間枠、MCP月間枠、モデル別統計
- 対応IDEにおけるGitHub Copilot拡張の導入検知
- 期間別グラフとプロバイダー絞り込み
- Codex／Claude Codeセッション開始時のタスクトレイ自動起動
- プロンプト本文や認証情報を保存しないローカルファースト設計

## ダウンロード

- `agent-meter-windows-x64-0.1.0.zip`
- `agent-meter-windows-x64-0.1.0.zip.sha256`

両方をダウンロードし、ZIPを展開する前にSHA256を照合してください。

## 注意事項

- Windows 10／11 x64向けです。
- 現在のバイナリにはコード署名がありません。
- Codexでは初回だけ`/hooks`でAgent Meterフックを信頼する必要があります。
- GitHub Copilotの個人向けリアルタイム残量は、公式APIがないため表示しません。
- ポート`127.0.0.1:4318`を別のアプリが使用している場合は起動できません。
